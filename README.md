## Description 


A mysql database and CMS-style Blog built using Model View Controller (MVC) paradigm.

## Usage 

Clone the repo, navigate to the root folder, run the following commands from the CLI:     

Run Test;    
`npm run test`     

Seed DB;   
`npm run seed`     

Start Server;     
`npm start server`
